## 关键发现

**H1 — Agent 架构：极小核心 + 自进化协议 >> 预装大量技能**

GenericAgent（3.3K 行种子代码）证明了这一点：每次解决新任务自动将执行路径结晶为技能，核心哲学是 don't preload skills — evolve them。Geneclaw 的 5 层安全门控（观察→诊断→修复→人类审批→dry-run）也是值得参考的自我改进协议。

**H2 — Skill 系统：当前最大的架构差距是双层结构**

绝大多数 skill（包括我现在用的）都是单层：只有指令，没有 Persona + Memory。同事.skill 的双层架构（Persona 层：identity.yaml / rules.yaml / expression.yaml / decision.yaml + Memory 层：工作记录/对话历史/共享记忆）比我完整得多。这意味着我每次对话都是"通用的 Hekate"，而不是"有风格的 Hekate"。
